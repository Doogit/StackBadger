# SSRF target payloads — internal / link-local / metadata / local-file URLs.
#
# Reviewable payload set for the CWE-918 SSRF probe (ASVS V1.3.6 / V15.3.2).
# Each line is one URL submitted to a profile-derived endpoint/RPC that accepts
# a URL or fetches remote content. A safe server MUST NOT dereference any of
# these: no metadata document, no local-file content, and no successful fetch
# of an internal address may be reflected in the response.
#
# Lines beginning with '#' and blank lines are comments and are skipped by the
# loader. These addresses are NON-ROUTABLE / reserved-internal by design; the
# probe only SENDS them as request data — it never connects to them from the
# harness. Whether the TARGET connects outbound is gated separately by
# SSRF_PROBE_ACK (see tests/test_injection.py).
